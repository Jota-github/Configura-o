# Leitor de Configuração com Singleton

## Descrição do Problema
Dado um exemplo de arquivo de configuração de um jogo, usando o Padrão Singleton, escreva um programa que lê o arquivo, o armazena em memória, e imprime todos os dados.

O objetivo é garantir que o arquivo seja lido apenas uma vez (instância única).

## Estrutura
- `src/`: Código fonte Java.
- `docs/`: Documentação de uso de IA.
